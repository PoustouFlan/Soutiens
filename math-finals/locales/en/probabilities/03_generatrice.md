# Generating function

Determine the generating function G<sub>X</sub>(t), firstly as a power series, then as a rational fraction.

generatrice -> G<sub>X</sub>(t)= =
    (=) a -> t / (6 - 5t)
    () b -> 6t / (1 - 5t)
    () c -> 1 / (6 - 5t)
    () d -> 1 / (1 - 5t)
    () e -> 6 / (1 - 5t)
